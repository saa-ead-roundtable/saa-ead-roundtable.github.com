This page links to resources maintained by the SAA Encoded Archival Description Roundtable hosted on Github.

User-Contributed EAD Stylesheets

A repository for the EAD user community to share XSLT stylesheets and related resources for the rendering of archival description expressed in EAD.

EAD 1.0 to EAD 2002 Conversion Package

Converts valid EAD 1.0 instances to EAD 2002 instances. This package was originally developed by Stephen Yearl.

EAD Cookbook

Markup recommendations, stylesheets, and software tools to render EAD finding aids. These packages were originally developed by Michael Fox.
